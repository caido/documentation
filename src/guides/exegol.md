## Native Support 

Caido now comes pre-installed in:

- **Free image** : `free`
- **Advanced images** :  `full`, `web`, `ad`, `nightly`
- **Private Images**

## Launching Caido

To launch Caido, just enter in you container terminal a following command and `Caido` desktop will appear.

```bash
	caido --no-sandbox
```

<img alt="Exegol banner." src="/_images/caido_on_exegol.png" />

And if you want the help  use `caido-cli` like this :

```
caido-cli --help
```
