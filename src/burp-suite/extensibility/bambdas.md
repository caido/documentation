# Bambdas

Burp Suite Pro Bambdas — in-app JavaScript snippets — and their Caido equivalents.

### Bambdas

Lightweight JavaScript snippets that run inside Burp for filtering, custom actions, and UI automation.

Use native **Workflows** for traffic filtering, transformation, and passive or active analysis — the closest equivalent to Bambdas that operate on requests and responses. For UI-level automation (custom buttons, panels, or actions), build a **plugin** with custom pages and commands through the plugin SDK. Workflows replace Bambdas' traffic logic; plugins replace Bambdas' UI automation.
